## CMS editor experiment

A little proof of concept for a travel blog hooked up with a cms

# Plans:
- Sidebar interface for editing (complete)
- Inline interface for editing (tbd)

# Technologies
- Nextjs - React
- Typescript
- Material Ui
- Styled components
- tinycolor2
- color thief
- React Redux Saga
- Firebase -> Auth and Firestore
- Unplash source
